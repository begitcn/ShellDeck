🔄 核心变更：融合本地终端 & SSH 为统一「会话列表」

  • 重构 移除模式切换器——不再有 SSH/本地终端的分段切换。两者共存于侧边栏
  • 统一 SSH 服务器按分组（生产/开发/个人），本地终端独立为一个紫色标记的「本地终端」分区
  • 无缝切换 点击侧边栏任一会话直接切换内容区——SSH 终端、本地终端、文件管理、监控都在同一视图层级
  • 融合 本地终端会话有 独立标签栏（类 iTerm2），支持多标签快速切换

🎨 视觉改进（A 布局 + C 配色）

  • 新增 分组彩色左边框：生产=红 / 预发布=橙 / 开发=绿 / 个人=蓝
  • 新增 连接芯片（connection-chip）——在 toolbar 直观显示当前连接状态
  • 新增 底部状态栏——连接状态、主机名、终端尺寸、延迟一目了然
  • 修复 文件管理器改为 Grid 布局，支持列排序、键盘导航、权限列
  • 修复 拖放上传区域从 overlay 改为条状提示，不影响文件列表操作
  • 新增 本地终端标签栏（多会话时顶部显示类似 iTerm 的 tab bar）
