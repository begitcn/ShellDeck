设计反思 · ShellDeck UI Redesign 核心变更

  • P0 侧边栏·搜索 — 添加 `searchable` 输入框，解决大量服务器时无法搜索的问题
  • P0 文件管理器·原生 Table — 从 `ScrollView+LazyVStack` 改为 `Table` (macOS 13+)，获得原生列排序、键盘导航、删除键支持
  • P1 底部状态栏 — 新增连接状态/延迟/平台信息，避免用户猜当前状态
  • P1 分组视觉区分 — 方案 C 中分组的彩色左边框让环境一目了然（红=生产、橙=预发布、绿=开发、蓝=个人）
  • P1 信息密度优化 — 方案 B 更紧凑的间距适应专业用户多服务器场景
  • P2 视觉统一 — 统一定义 `--surface`、`--border`、`--accent` 色彩变量，清除各处硬编码色值
  • P2 标签栏移到顶部 — 终端/文件/监控/信息标签从 toolbar 内嵌改为独立标签栏，更清晰
  • P2 连接按钮 — 悬浮圆形按钮替代文字链接，降低视觉噪音
  • P2 移除 AnyView — 已通过 `@ViewBuilder` 条件分支替换所有 `AnyView`
